# Create a multipanel theme compatible with ggplot2 4.0.0 S7 system

Creates a multipanel theme that is compatible with the new ggplot2 4.0.0
S7 system. This function creates a fresh theme object using the new S7
system instead of relying on the saved S3 theme object.

## Usage

``` r
sccomp_theme()
```

## Value

A `ggplot2` theme object compatible with ggplot2 4.0.0
