# Deprecated interval-plot aliases

These soft-deprecated aliases forward to
[`sccomp_plot_intervals_1D()`](https://mangiolalaboratory.github.io/sccomp/reference/sccomp_plot_intervals_1D.md)
and
[`sccomp_plot_intervals_2D()`](https://mangiolalaboratory.github.io/sccomp/reference/sccomp_plot_intervals_2D.md).
Use those functions directly instead.

## Usage

``` r
plot_1D_intervals(...)

plot_2D_intervals(...)
```

## Arguments

- ...:

  Arguments passed on to
  [`sccomp_plot_intervals_1D()`](https://mangiolalaboratory.github.io/sccomp/reference/sccomp_plot_intervals_1D.md)
  or
  [`sccomp_plot_intervals_2D()`](https://mangiolalaboratory.github.io/sccomp/reference/sccomp_plot_intervals_2D.md).

## Value

A `ggplot` object, as returned by the target function.
